Drag and Dropload

Wednesday, December 8th, 2010

Uploading multiple images at once has never been easier. It’s now possible to simply select a bunch of images in a folder, and drag them on over to Imgur. It works on both the homepage, and inside your account “images” page.


Keep in mind, in order for this to work, you need to be using the latest version of either Mozilla Firefox or Google Chrome.
