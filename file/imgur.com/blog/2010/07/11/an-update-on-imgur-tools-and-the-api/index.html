An update on Imgur tools and the API

Almost every day someone asks about being able to upload images into their account using one of the tools. I figured that since this question is so relevant, it deserves its own blog post.

This functionality is coming very soon. The means to do so is being worked on right now, and so are some really exciting new tools that you can use to manage your entire account with.

Here’s where this post gets a little technical. A new version of the API — or at least a beta version of it — will be released in the coming weeks. It will expose almost all of the Imgur service via a standardized programmatic interface, be completely compatible with the current version and will add on a full set of commands to use for managing accounts. This includes uploading into accounts, creating albums, etc. Authentication will be done using OAuth, and will work very similar to the Twitter API.

The reason I’m announcing all of this before it’s released is because I want your input. If you’re a developer, now is the time to submit feature requests, ideas, thoughts, or anything else on how it should work, or things you would like to see included with it. If there’s something you don’t like about the current API, or if you just have a neat idea about something that would be cool to include, then I’d love to hear it. I’ve created a special area in the forums dedicated for this, so please, share your thoughts!
