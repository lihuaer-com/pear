Every other comment is just me trying to help, answering questions or sometimes even to be funny. If you don't want this bot to comment on your posts write me a message and i'll add your name to a blacklist. Please notice that if you change your username the bot will comment again. If you are tired of seeing the bots comments, check http://imgur.com/gallery/QEEC1Nl ################################### WTF is this ? This bot detects when an image was seen for the first and the last time by generating hashes describing every image. The strange stars represent how similar the two titles are via the levenshtein distance (How many chars have to be changed/inserted/removed to change text A to text B). ☆☆☆☆☆ means no similarity ★★★★★ means its almost or exact the same title ################################### ################################### Stats and conditions Following conditions have to be fulfilled at the moment for the bot to post: - Both images are posted by a different user - The time between both posts is <= 60 days OR The title similarity is >= 60% - If the title contains 'mrw' or 'mfw' or starts with 'when' the title similarity needs to be >= 60% - The repost need to have at least 40 points Oldest indexed image at the moment: - 3.40900249594 years Images indexed: - 255073 User on blacklist: - 19 ################################### ################################### For programmer: The bot is coded in python. Used hashes are: dHash and aHash with 256 bit to decrease false positives. http://www.hackerfactor.com/blog/index.php?/archives/2013/01/21.html Animated images are only compared by crc. * I will open source the code, but first i need to add some features and clean it up* TODO: - Galleries (NOT doable within api limits) - Play with pHash - Add notification features - Work on the userscript - reaction pic filter why ? because i like coding ###################################
