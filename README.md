Pear
=========
基于Toropress衍生而来的图站源码.是Lihuaer.com的开发代号

## 图片分享社区演示网站


这是一个分享图片为主的社区系统~
基于[Beego](https://github.com/astaxie/beego)与[xorm](https://github.com/lunny/xorm)开发的GOLANG图片社区系统网站，从toropess源码衍生而来!


[访问 梨花儿社区](http://www.lihuaer.com/)
